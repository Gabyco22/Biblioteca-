# Biblioteca-

es Descripción
Este proyecto es un sistema de gestion de biblioteca dessarrollado en Python + SQLite.
Permite administrar libros, socios y prestamos de manera sincilla desde la misma consola.
Fue diseñado con un enfoque modular, separando la conexion a la base de datos de la logica del programa, para una mayor claridad y mantenibilidad.

✨ Funcionalidades
- Gestión de libros: agregar, listar, contar y filtrar por año.
- Gestión de socios: registrar y consultar socios.
- Gestión de préstamos: registrar préstamos y devoluciones, consultar libros prestados.
- Validaciones: evita duplicados y préstamos de libros ya ocupados.
- Menú interactivo: interfaz simple en consola para navegar entre opciones.
🛠️ Tecnologías utilizadas
- Python 3
- SQLite3

🚀 Cómo ejecutar
- Clonar el repositorio:
https://github.com/Gabyco22/Biblioteca-.git

- Ejecutar el programa:
python BibliotecaG.py



🇬🇧 Description
This project is a Library Management System built with Python + SQLite.
It allows managing books, members, and loans through a simple console interface.
The system is modular, separating database connection from program logic for clarity and maintainability.
✨ Features
- Books management: add, list, count, and filter by year.
- Members management: register and list members.
- Loans management: register loans and returns, check borrowed books.
- Validations: prevents duplicates and borrowing of already loaned books.
- Interactive menu: simple console interface to navigate options.
🛠️ Technologies
- Python 3
- SQLite3
🚀 How to run
- Clone the repository:
https://github.com/Gabyco22/Biblioteca-.git
- Run the program:
python BibliotecaG.py
